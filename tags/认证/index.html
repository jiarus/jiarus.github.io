AWS DVA-C02 备考指南

目标：用可执行的学习计划通过 AWS Certified Developer - Associate (DVA-C02)。

考试概览

  • 考试时长：130 分钟
  • 题型：单选/多选
  • 重点方向：应用开发、部署、监控、身份与权限、基础架构上的最佳实践

能力版图与权重

建议按以下模块分层复习：

  1. 部署 (Deployment)
    • CodeDeploy / CodePipeline / CodeBuild
    • Blue/Green 与滚动发布
  2. 安全 (Security)
    • IAM 权限边界、STS、角色切换
    • KMS、Secrets Manager、Parameter Store
  3. 开发与集成 (Development with AWS Services)
    • Lambda / API Gateway / DynamoDB / SQS / SNS
    • SDK 重试、签名、分页
  4. 故障排查与优化 (Troubleshooting and Optimization)
    • CloudWatch / X-Ray
    • 常见延迟与错误定位

学习路径（4 周）

第 1 周：核心服务 + 基础安全

  • IAM、STS、KMS
  • Lambda、API Gateway、DynamoDB
  • 每天 20 题小测，整理错题

第 2 周：部署与交付

  • CodePipeline、CodeBuild、CodeDeploy
  • Elastic Beanstalk
  • 重点理解部署策略与回滚

第 3 周：消息与事件驱动

  • SQS、SNS、EventBridge
  • Step Functions
  • 结合 Lambda 的触发与重试

第 4 周：综合演练

  • 全真模拟 2-3 套
  • 错题回看，补齐薄弱环节

高频考点清单

  • DynamoDB 分区键/排序键、RCU/WCU、GSI/LSI
  • Lambda 触发器、并发、冷启动
  • API Gateway 鉴权方式（IAM/Cognito/自定义）
  • SQS FIFO 与 Standard 的差异
  • KMS vs Secrets Manager vs Parameter Store
  • CloudWatch Metrics/Logs/Alarms
  • X-Ray trace 与服务地图

备考资料建议

  • 官方考试指南与样题
  • 控制台实践：每天 30-60 分钟动手
  • 题库用于查漏补缺，不要只刷题

实战建议

  • 把常见错误码、重试策略做成卡片
  • 每道错题写出“正确的 AWS 服务/配置”
  • 做 1 次完整的 Serverless 小项目（API Gateway + Lambda + DynamoDB）

考前 48 小时

  • 不再刷新题，重点复盘错题
  • 复查 IAM、DynamoDB、Lambda、API Gateway
  • 保证睡眠，按考试时间进行模拟

结语

DVA-C02 强调“开发者视角”，理解服务的使用方式与限制比死记更有效。按计划稳步推进，基本都能顺利通过。
